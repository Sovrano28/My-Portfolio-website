WhatsApp UI Clone

Live site
A fully functional and responsive replica of WhatsApp user interface.
Type: Social App UI
Stack: HTML, JavaScript, and Tailwind CSS.
About Project

This project is a faithful recreation of the WhatsApp Web user interface, demonstrating a strong command of core frontend technologies. Built entirely from scratch with HTML, CSS, and vanilla JavaScript, it leverages Tailwind CSS for rapid, responsive, and maintainable styling. The interface is fully interactive and includes the standout ability to switch themes on the fly. With a single click, you can move between a clean light mode and a comfortable dark mode, showcasing dynamic UI manipulation and attention to user experience

Responsive Design

This WhatsApp UI clone is designed to be fully responsive, ensuring optimal user experience on both mobile and desktop devices. The layout, elements, and components have been carefully crafted to adapt seamlessly to different screen sizes

Light and Dark themes

The app has light and dark mode features and can be changed by going to Settings just like in your regular WhatsApp for desktop or mobile.
